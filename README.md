# Messenger App

Simple web application which allows users to send messages (texts, images, videos, ect) with connected people.

## Table of content
- [Technologies](#technologies)
- [Setup](#setup)


## Technologies
- `Node.js`
- `Express.js`
- `MongoDB`
- `Mongoose`
- `jQuery`

## Setup
To run this project, install it locally using `npm`
````
$ npm install
$ npm run dev
````
